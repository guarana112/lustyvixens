# LustyVixens Free Website Stack

Open `index.html` in a browser or upload this folder to any static host.

Included:

- Creator upload form with Paste Link and Upload File modes
- Local admin review queue
- JSON export backup
- Basic analytics dashboard
- Starter Terms, Privacy, and DMCA pages
- Email sequence templates
- Domain portability checklist
- GitHub Actions backup reminder

This is a free launch foundation. It does not process real files, download videos, or upload to Voe until you connect a backend.
